# Contributing

- Mau nambahin term? Gas aje ye PR.
- Mau diskusi dulu? Gas aje ye bikin Issue.

Tiap term formatnya:

```
### Nama Term
- <ins>Sejak</ins>: Mmm 'YY
- <ins>Kategori</ins>: konsep / AI model / tool / perusahaan / infra
- <ins>Family</ins>: AI Family / DevOps Family / Miscellaneous
- <ins>Konteks</ins>: jelasin konteksnya, bukan definisi. santai aja
> Link: <url>
```

Jangan lupa tambahin di tabel kategori dan family di atas.
